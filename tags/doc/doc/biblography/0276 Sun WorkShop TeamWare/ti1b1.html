Checking In a File

Related Topics:

Checking Out a File

To check in a file:

1. In the Versioning Window, navigate to the directory where the files are stored.

2. Click the icon for file you want to check in. You can Shift-click to select multiple files.

Files that are checked out have a red check mark on the icon.

3. Choose Commands Check In.

4. In the Check In dialog box, type a comment.

5. Click OK.

Sun WorkShopTM Teamware saves your comments and checks in the file. To integrate the changes to the parent workspace, see Integrating Changes by Putting Back Files.
