File Menu

The File menu in the Configuring Window provides the following commands:
Load Workspaces Opens the Load Workspaces Dialog Box, where you can select a workspace to load into the Workspace pane.
Create Workspace Opens the Create Workspaces Dialog Box, where you can name the new workspace to be created.
Unload Workspaces Removes the selected workspace from the Workspace pane.
Exit Closes Configuring.
