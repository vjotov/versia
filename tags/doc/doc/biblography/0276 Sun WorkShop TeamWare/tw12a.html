Workspace History Viewer Window

Related Topics:

Viewing and Customizing Workspace History

Use the Workspace History Viewer window to view the history of the selected workspace. To open the History Viewer window, in the Configuring Window, choose Workspace View History.

This window contains the following items:

Directory Box

Type the name of the workspace here.

Transaction Report Pane

Displays transaction information. See Adding and Removing Transaction Information and Sorting Transaction Information.

Filter Check boxes

Use these check boxes to sort transaction information. See Filtering Transaction Information.

Transaction Details Pane

Displays the following transaction details: start date and time, end date and time, parent, child, directory, and Integration Request ID (if applicable).

Comments Pane

Displays the comments that are associated with each transaction. See Searching Comments or the Command Log

Command Log Pane

Displays the list of commands that are executed on the workspace.

If the Workspace History Viewer window is too large, you can collapse sections of it. See Collapsing or Expanding Panes.

Buttons
Close Closes the window.
Help Displays online help for this window.
