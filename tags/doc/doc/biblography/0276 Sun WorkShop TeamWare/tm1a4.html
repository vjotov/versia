Navigate Pop-Up Menu

The Navigate pop-up menu is displayed when you click on the right mouse button while in one of the Merging file panes.
Next Unresolved Difference Moves you to the next unresolved difference.
Previous Unresolved Difference Moves you to the previous unresolved difference.
Selected Difference Moves you to the previous unresolved difference.
First Difference Moves you to the previous unresolved difference.
Find Opens the Find Dialog Box
Goto Opens the Goto Line Dialog Box
What Line Displays the number(s) of theselected line(s) and the total number of lines in the file.
