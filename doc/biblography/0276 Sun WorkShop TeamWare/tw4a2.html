Bringover/Putback Options Tab

Use the Bringover/Putback Options tab to set options for bringovers and putbacks. To open the Bringover/Putback Options tab, in the Configuring Window, choose Options Configuring and select the Bringover/Putback tab.

This tab contains the following items:.
Transaction File List: Autoload check box Reads the CodemgrWsdata/args file and loads it into the File Pane whenever a new workspace is selected. Deselect this property when you want to use the same file list for a number of transactions involving different workspaces. The default is on.
Transaction Output: Auto Display check box Displays the Transaction Output dialog box during Bringover, Putback, and Undo transactions. If this is not selected, you must activate the Transaction Output dialog box using the Show Output button. The default is on.
Putback: Auto Bringover Update check box Invokes a bringover when a putback determines that files have changed in the parent. The default is off.
Bringover/Putback: Warn Comment Reusing check box Displays an alert reminding you that the comments are the same as the last Bringover/Putback. The default is on.
