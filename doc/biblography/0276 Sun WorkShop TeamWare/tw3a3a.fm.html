Notification Entry Dialog Box

Use the Notification Entry dialog box to set up automatic notification for transactions. To open the Notfication Entry dialog box, in the Configuring Window, choose Workspace Properties, select the Notification Tab and click Create Entry/Edit Entry.

This dialog box contains the following items:

Workspace Display

Displays the current workspace.

Mail To Text Box

Type the e-mail address(es) of those people (or groups) you wish to notify.

Events Checkboxes

Select which transactions will generate an e-mail notification:
bringover-from Bring over files from this workspace.
bringover-to Bring over files to this workspace.
putback-from Put back files from this workspace.
putback-to Put back files to this workspace.
undo Undo commands executed in this workspace.
workspace-delete Delete this workspace.
workspace-move Move this workspace.
workspace-reparent Reparent this workspace.
workspace-reparent-to Reparent other workspaces to this workspace.

Files Buttons
All Sends notification when the selected transactions occur for any file in the workspace.
Specify Sends notification when the selected transactions occur for only the files listed.
Add to List Opens the Add Files Dialog Box, where you can select files.
Delete Deletes the selected files and directories.
Delete All Deletes all of the files and directories.

Buttons
OK Applies the change and closes the dialog box.
Apply Applies the change without closing the dialog box.
Reset Reverses the most recent change.
Cancel Closes the dialog box without applying changes.
Help Displays the online help for this dialog box.
