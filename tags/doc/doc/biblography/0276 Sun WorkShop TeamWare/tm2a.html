Open Files Dialog Box

Use the Open Files to dialog box select the files for your merge. To access the Open Files dialog box, in the Merging Window, choose File Open or click on the Open button.

At a minimum, you need to enter a file for the left pane and a file for the right pane. Each box is followed by an ellipsis button, which you can click to open a file chooser, where you can select a file or directory.

This dialog box contains the following components:

Directory Box

Displays the current directory.

Left File Box

Type the name of the file to be loaded into the Child Pane.

Right File Box

Type the name of the file to be loaded into the Parent Pane.

Ancestor File Box

Type the name of an earlier cop.y of the file to be used when merging

Output File Box

Type the name to use when saving the output file.

Buttons
Open Opens the selected files and directories in the Merging window.
Cancel Closes the Open Files dialog box without opening any files.
Help Displays help for this dialog box.
