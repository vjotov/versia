Creating an Empty Workspace

Related Topics:

Creating a Workspace From Existing Files Creating a Child Workspace

To create an empty workspace:

1. In the Configuring Window, choose File Create Workspace.

2. In the Create Child Workspace Dialog Box, enter a name in the Workspace Directory text box. If the directory does not exist, Configuring will create it for you.

Your new workspace contains a directory named Codemgr_wsdata. This directory contains the metadata files for this workspace. (Metadata files are files that contain data about the data in workspaces.)
