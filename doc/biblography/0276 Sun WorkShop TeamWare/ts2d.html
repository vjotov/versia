Giving a Workspace a Descriptive Name

Related Topics:

Moving or Renaming a Workspace

To give a workspace a descriptive name:

1. In the Configuring Window, load the workspace by choosing File Load Workspaces.

2. Select the workspace by clicking on it.

3. Choose Workspace Properties.

4. In the Workspace Properties Dialog Box, choose the Description Tab.

5. In the Name text box, type a descriptive name for your workspace.

This name appears as a label for the workspace.

6. In the Description text box, type a longer description of your workspace.

This information is stored in the Codemgr_wsdata/description file.

7. Click OK.
Note - Clicking Load displays description information from the description file. It does not save any information.

To view descriptive names:
   In the Configuring window, choose View Descriptive Names,

or

   Use the workspace descr command at the command line.
