Recovering Files From a Freezepoint File

To re-create a workspace from a freezepoint file:

1. In the Configuring Window, choose TeamWare Freezepointing to open the Freezepointing window.

2. You can accept the default name of freezepoint.out or type the name of the freezepoint file in the Freezepoint File box.

3. Click the browse button to the right of the workspace box or type the name of the workspace in the Workspace box.
Note - Because a freezepoint file is only a list of differences, you must have the original workspace (or its parent or child) to extract from a freezepoint file.

4. If you want to create a workspace from the extracted files, click the Create Workspace check box.

By default, the recreated workspace does not contain any SCCS history files. It contains only the files from which the orginal files were created.

5. Click Add to add files and directories to the Directories and Files list.

6. Click Extract.
