Viewing and Customizing Workspace History

Related Topics:

Workspace History Viewer Window Viewing File History

The Workspace History Viewer allows you to see a listing of transactions for a selected workspace.

To view the history of a workspace:

1. In the Configuring Window, click on a workspace to select it.

2. Choose Workspace View History.

You can modify the way the Workspace History Viewer window looks in several ways. See:

  • Adding and Removing Transaction Information

  • Sorting Transaction Information

  • Filtering Transaction Information

  • Searching Comments or the Command Log

  • Collapsing or Expanding Panes
