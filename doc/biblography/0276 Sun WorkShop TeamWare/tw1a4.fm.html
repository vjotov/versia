The Actions menu in the Configuring Window opens various tabs in the Transactions dialog box. Use these tabs to do bringovers and putbacks, to resolve files, and to undo transactions.

In the Transactions dialog box, you'll find the following tabs
Bringover Create Tab Creates new workspaces using a bringover.
Bringover Update Tab Updates current workspaces using a bringover.
Putback Tab Puts back changes to workspaces.
Resolve Tab Resolves conflicts in files brought over from the parent workspace.
Undo Tab Reverses a bringover or putback.
