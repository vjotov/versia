Add Files Dialog Box

Use the Add Files dialog box to choose files and directories to include in the freezepoint file. To open the Add Files dialog box, in the Freezepointing Window, click the Add Files button.

This dialog box contains the following items:

Directory Display

Shows the currently selected directory.

File Pane

Lists the contents of the currently selected directory

Name Box

Allows you to type file names.

Buttons
Add Files to List Adds the selected files to the File pane.
Load Directory Displays the contents of the directory entered in the Name box.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
