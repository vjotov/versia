Update Nametable Dialog Box

Use the Update Nametable dialog box to update the record of workspace's names and the names of the files they contain. To open the Update Nametable dialog box, in the Configuring Window, choose Workspace Update Nametable.

This dialog box contains the following items:

Directory Display

Displays the name of the current workspace.

Buttons
Update Nametable and Files' Name Histories radio button Lets you update both the Nametable and the files' name histories.
Update Nametable Only radio button Lets you update the nametable only and does not change the files' name histories.
OK Applies the changes the closes the dialog box.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
