Notification Tab

Use the Notification tab to add, delete, and change notification entries for the specified workspace. Notification messages are triggered by the events and sent to the specified mail address. To open the Notification tab, in the Configuring Window, choose Workspace Properties and select the Notification tab.

This tab contains the following items:

Notificiations Pane

Displays current notification entries for the specified workspace. The mail address to which notification messages are sent is listed on the left and the events that trigger notification are listed on the right.

Buttons
Load Loads the notification entries from the Codemgr_wsdata\notification file for the workspace specified in the Workspace box.
Delete Deletes lock entries selected in the Locks pane.
Create Entry Opens the Notification Entry Dialog Box, where you can set a new notification.
Edit Entry Opens the Notification Entry Dialog Box, where you can edit the selected notification.
