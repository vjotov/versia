Create Child Workspace Dialog Box

Use the Create Child Workspace dialog box to create empty child workspaces. To open the Create Children dialog box, in the Configuring Window, choose Workspace Create Child.

This dialog box contains the following items:

Parent Workspace Directory Text Box

Type the path name of the parent workspace directory to create a child from.

Child Workspace Directory Text Box

Type the path name of the new workspace you want to create.

Check Box
Do Bringover Check this box to Bringover the files in the parent workspace when you create the child workspace. The default is to create an empty child workspace.

Buttons
OK Applies the changes and closes the dialog box.
Cancel Closes the dialog box without making changes.
Help Displays the help for this dialog box.
