Check In New Dialog Box

Use the Check In New dialog box to check in new files. In this dialog box, you bring files under SCCS control so Sun WorkshopTM TeamWare can recognize them. To open the Check In New dialog box, in the Versioning Window, choose Commands Check In New.

This dialog box contains the following components:

Filter Box

Type a text string to limit which files which appear. You can use wildcards such as "*" and "?".

Directory Box

The current directory is displayed here. You can type another the path name here.

File List

Lists the files in the current directory not under SCCS.

Initial Comment Pane

Type the comments you want to add to the SCCS file history.

Buttons
OK Applies the changes and closes the dialog box.
Apply Applies the changes without closing the dialog box
Clear Clears the comments from the Initial Comment box
Cancel Closes the dialog box without making changes.
Help Displays help for this dialog box.
