Deleting Workspaces

Related Topics:

Turning a Workspace Into a Regular Directory

Caution - It is important that you use Sun WorkShopTM TeamWare to move, rename, or delete workspaces. If you use any other method, such as directly manipulating the file system, Sun WorkShop TeamWare cannot maintain the parent-child relationships of your workspaces.

To delete an entire workspace (including the files and the metadata files):

1. In the Configuring Window, load the workspace by choosing File Load Workspaces.

2. Select the workspace by clicking on it once. You can Shift-click to select multiple workspaces.

3. Choose Workspace Delete.

4. Click OK in the Delete Workspaces Dialog Box.

The workspace you deleted is no longer displayed in the Configuring window.
