Workspace Menu

The Workspace Menu in the Versioning Window contains the following commands.
Delete Files Opens the Delete Confirmation Dialog Box, where you can confirm you want to delete the selected files.
Move Files Opens the Move Confirmation Dialog Box, where you can confirm you want to move the selected files and select a new destination directory.
Find Opens the Find Dialog Box, where you can scan the entire workspace for checked out files or for history files that contain unmerged deltas.
