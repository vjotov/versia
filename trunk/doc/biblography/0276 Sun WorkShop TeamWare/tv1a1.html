File Menu

To access the File Menu, in the Versioning Window, choose File. The File Menu contains the following commands:
Load Loads a directory into the Directory pane. The submenu contain directories you have already visited.
Edit Opens the selected file in a text editor. To change your text editor, see Changing Your Editor.
Latest Diffs Opens the Merging Window so you can view the differences between the latest edits of a selected file and the previously checked in version.
File History Opens the File History Window, where you can view the history of the selected file.
File Info Opens the SCCS File Properties Dialog Box, where you can change SCCS flags.
Exit Closes the Versioning window.
