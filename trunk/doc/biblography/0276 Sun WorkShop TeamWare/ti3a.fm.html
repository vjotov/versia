Viewing File History

Related Topics:

Viewing and Customizing Workspace History

In Versioning, you can see who made changes to a file and when the changes were made.

To view the history of a file:

1. In the Versioning Window, click a file to select it.

2. Choose File File History. This displays the File History Window.

3. When you want to view another delta of the same file, click a delta (represented by a number, for example, 1.2) in the History Graph pane.

History Viewer Symbols

  • Red check marks indicates the file is checked out.

  • Solid lines indicate the default deltas path.

  • Arrows indicate merged deltas.

  • Crossed out deltas indicate the delta is unmerged.
