Help Menu

The Help menu in the Configuring Window provides the following commands:
Contents Opens the Sun WorkShopTM TeamWare online help.
Readme Displaying the readme for the current version of Sun WorkShop TeamWare.
About Documentation Connects to About Sun WorkShopTM 6 Documentation.
Send Comments Opens a comment box in which you can send your comments about Sun WorkShopTeamWare to the Sun WorkShop TeamWare developers at Sun Microsystems, Inc.
About Sun Workshop TeamWare Opens the About box, which contains information about the current version of Sun Workshop TeamWare.
