Load New Menu Defaults Tab

Use the Load New Menu Defaults tab in the Options Dialog Box to add pointers to your favorite directories to the Versioning File Load menu. Instead of typing in a long path in the Directory Box, you can select the path from the File Load menu. To open the Load New Menu Defaults tab, in the Versioning Window, choose View Options and select the Load Menu Defaults tab.

This tab has the following components:

Customized Path List Box

List your customized paths after they are created.

Options Buttons
Add Add the selected path to the File Load menu.
Delete Delete the selected path from the File Load menu.
Edit Add or change the selected path.

Pathname Box

Type the full path of the directories you use most often.

Menu Label Box

Type the name you want displayed on the File Load menu.

Buttons
OK Applies the changes and closes the dialog box.
Apply Applies the changes without closing the dialog box.
Reset Cancels current changes.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
