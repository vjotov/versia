Turning a Workspace Into a Regular Directory

Related Topics:

Deleting Workspaces

To change a workspace into a regular directory:

1. In the Configuring Window, load the workspace by choosing File Load Workspaces.

2. Select the workspace by clicking on it.

3. Choose Workspace Delete.

4. In the Delete Workspaces Dialog Box, click the button labeled "Delete Codemgr_wsdata Directory only."

5. Click OK.

The workspace is no longer displayed in the Configuring window.
