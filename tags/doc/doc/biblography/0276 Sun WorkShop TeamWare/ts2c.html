Moving or Renaming a Workspace

Related Topics:

Workspaces Defined Rename Dialog Box
Deleting Workspaces

Caution - It is important that you use Sun WorkShopTM TeamWare to move, rename or delete workspaces. If you use any other method, such as directly manipulating the filesystem, Sun WorkShop TeamWare cannot maintain the parent-child relationships of your workspaces.

To rename or move a workspace:

1. In the Configuring Window, load the workspace by choosing File Load Workspaces.

2. Select the workspace by clicking on it.

3. Choose Workspace Rename.

4. In the Rename Dialog Box, type the new name or location for your workspace.

5. Click OK.

The workspace with the new name or location is displayed in the Configuring window.
