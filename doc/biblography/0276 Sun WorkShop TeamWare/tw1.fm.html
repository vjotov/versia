Configuring Defined

The Configuring window displays workspaces and provides access to the commands you use to manage workspaces. The Configuring window contains the following components:
File Menu Provides commands for loading, unloading, and creating workspaces.
View Menu Provides commands to change how the workspaces appear in the workspace pane.
Workspace Menu Provides commands for managing workspaces.
Actions Menu Opens the Transactions Dialog Box , which provides commands for synchronizing files (using bringovers, updates, and putbacks).
Options Menu Provides commands for setting the workspace and configuring options.
TeamWare Menu Provides commands to start other Sun WorkShopTM TeamWare tools (Versioning, Merging, and Freezepointing).
Help Menu Provides commands to display help and other release documentation.
Workspace Pane Displays a visual graph of the workspaces you have loaded.
