Freezepoint Basics Automatically Generating Freezepoints

Use the Freezepointing tab to select the times you want Sun WorkShopTM TeamWare to create freezepoint files for the specified workspace. To open the Freezepointing tab, in the Configuring Window, choose Workspace Properties and select the Freezepointing Tab.

This dialog box contains the following items:

Buttons

Determine when you want freezepoint files created:

  • Before bringover

  • After bringover

  • Before putback

  • After putback

  • Before undo

  • After undo

  • Before resolve

  • After resolve
