Reparent Dialog Box

Related Topics:

Reparenting A Workspace

Use the Reparent dialog box to change a workspace's parent. To open the Reparent dialog box, in the Configuring Window, choose Workspace Reparent.

This dialog box contains the following items:

Parent Workspace Directory Text Box

Select or enter the absolute path name of the new parent.

Child Workspace Directory Text Box

If a child workspace is already loaded, it will appear here. If not, or you wish to reparent another child workspace, select it or enter the path name here.

Buttons
OK Applies the changes and closes the dialog box.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
