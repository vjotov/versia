Use the Versioning window to view, check in, and check out files.

This window displays the directories and files of your project. The file folder icons are directories; the piece-of-paper icons are individual files. TeamWare displays all directories, but only displays files that are under SCCS control. (See Adding Files to an Existing Workspace.)

When a file is checked out, there is a red check mark on the file icon.

File Menu Provides commands to manage files.
View Menu Provides commands to control and update the icons shown in the directory pane.
Commands Menu Provides commands to check files in and out.
Workspace Menu Provides commands to find, delete, or move files to SCCS control.
TeamWare Menu Provides commands to start other TeamWare services.
Help Menu Provides commands to display online help and release documentation.
The Versioning window contains the following components:

Directory Box

Shows the directory currently displayed. You can type in another path, or use the pull-down list to select a path you have already visited.

Directory Pane

Displays the icons of of directories and files.
