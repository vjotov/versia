Delete Confirmation Dialog Box

Related Topics:

Deleting Workspaces

The Delete Confirmation dialog box allows you to confirm deletion of files. To open the Delete Confirmation dialog box, in the Versioning Window, choose Workspace Delete Files.

This dialog box contains the following items:

Workspace Pane

Lists the workspaces you want to delete.

Buttons
OK Deletes the file(s) and closes the dialog box.
Cancel Closes the dialog box without deleting files.
Help Displays help for this dialog box.
