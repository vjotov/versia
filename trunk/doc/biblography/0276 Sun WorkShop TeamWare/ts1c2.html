Creating a Workspace From Existing Files

Related Topics:

Creating a Child Workspace

If you have a directory containing the files that you want to make into a workspace:

1. In the Configuring Window, choose File Create Workspace.

2. Click the Browse button to open the file chooser.

3. In the file chooser, click the directory to select it.

4. Click OK.

5. In the Create Directory dialog box, click OK to create the workspace.

The workspace is created and it is loaded into the Configuring window. Configuring creates a Codemgr-wsdata directory inside your existing directory. If your files are not already under SCCS version control, you need to check them in (See Putting Files Under SCCS Version Control).
