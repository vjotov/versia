Reversing Changes to a File

Related Topics:

Checking In a File Integrating Changes by Putting Back Files

Sun WorkshopTM TeamWare provides a way to return a file to the state it was in before it was last checked out. This is useful if you have made changes to a file, but do not want to check in those changes.

To reverse the changes made to a checked-out file:

1. In the Versioning Window, navigate to the directory where the checked-out file is stored.

The icon of the file has a red check mark on it, indicating it is checked out.

2. Click on the file to select it. You can Shift-click to select multiple files.

3. Choose Commands Uncheckout.

This reverses the most recent changes to the document.
