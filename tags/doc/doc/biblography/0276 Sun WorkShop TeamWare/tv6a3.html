Find Dialog Box

The Find dialog box finds checked out or unmerged files in the selected workspace. To open the Find dialog box, in the Versioning Window, choose Workspace Find.

This dialog box contains the following items:

Workspace Pane

Lists the current workspaces. You can type in the path of another workspace.

Find Checked out Files Radio Button

Select to search the workspace for checked out files.

Find Unmerged Files Radio Button

Select to search the workspace for unmerged files.

Bring up History window Check Box

Select to open the Versioning History window when you double-click on the selected file or when you click on GoTo.

Buttons
Start Starts searching the workspace.
Stop Stops searching the workspace.
GoTo Brings up the File History Window for the selected file.
Close Closes the window.
