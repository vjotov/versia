Tools Properties Dialog Box

Use the Tools Property dialog box, where you can set properties for using Configuring, such as the default double click action. To open the Tool Properties dialog box, in the Configuring Window, choose Options Configuring.

This window contains the following components:

Tabs
Codemanager Options Tab Sets properties for Configuring.
Bringover/Putback Options Tab Sets properties for Bringover and Putback transactions.
Resolve Options Tab Sets properties to control Resolve transaction behavior.
History Viewer Options Tab Sets properties for the History Viewer.

Butttons
OK Applies the change and closes the dialog box.
Apply Applies the change without closing the dialog box.
Reset Cancels all changes and resets fields.
Cancel Closes the dialog box without applying changes.
Help Diplays the help for this dialog box.
