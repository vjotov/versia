Options Dialog Box

Use the Options dialog box to set options for the Versioning Window. To open the Options dialog box, in the Versioning Window, choose View Options.

In the Options dialog box, you'll find the following tabs:
General Tab Lets you modify what you view in the main filelist and History Graph, modify the mouse double-click action, and change your default editor.
Customized Menu Tab Lets you add your own customized commands.
Load New Menu Defaults Tab Lets you create an alias for workspace names. This is useful if you have a workspace with a long path.
