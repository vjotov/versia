Putting Files Under SCCS Version Control

Related Topics:

Creating a Child Workspace

For the Sun WorkShopTM TeamWare configuration management tools to be able to recognize files, those files must be under SCCS version control.

To put files under SCCS version control:

1. In the Configuring Window, load the workspace you want work with by choosing File Load Workspaces.

If the directory that contains your files is not already a workspace, see Creating a Workspace From Existing Files.

2. Click the workspace to select it.

3. Choose TeamWare Versioning.

4. In the Versioning Window, choose Commands Check In New.

5. In the Check In New Files dialog box, select the files you want to place under SCCS version control. You can Shift-click to select multiple files.

6. Click OK. The files you have checked in are displayed in the Versioning window.
