Load Workspaces Dialog Box

Use the Load Workspaces dialog box to load workspaces. To open the Load Workspaces dialog box, in the Configuring Window, choose File Load Workspaces.

This dialog box contains the following items:

Directory Name

Displays the current directory.

Directory Display

Displays the directories in the current directory. Workspaces are yellow folders with a workspace icon on them.

Name Text Box

Type the name of the workspace you want to load in this box.

Buttons
Load Workspaces Loads the workspace selected in the Directory Display or typed in the Name Text Box into the Configuring window.
Load Directory Opens the selected directory.
Close Closes the dialog box.
Help Displays the help for this dialog box.
