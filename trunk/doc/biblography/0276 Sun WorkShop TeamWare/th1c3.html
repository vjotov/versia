By default, all transactions appear in the Transaction Information pane and the only filter option you have is User.

To add filter options to the Workspace History Viewer window:

1. In the Configuring Window, click a workspace to select it.

2. Choose Workspace View History.

3. In the Workspace History Viewer Window, choose View Options.

4. Click the check boxes for the filters that you want to appear in the History Viewer window.

5. Click OK.

To filter transaction information:

1. In the Configuring Window, click a workspace to select it.

2. Choose Workspace View History.

3. Click the filter check box.

4. Type the user, host, or release you want to filter by, or select an Operation or Exit Status.

5. Click Apply Filter.

You can combine filters by selecting more than one check box.
