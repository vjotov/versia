Moving or Renaming a Workspace

Use the Rename dialog box to rename or move workspaces. To open the Rename dialog box, from the Configuring Window, choose Workspace Rename.

This dialog box contains the following items:

Old Workspace Name Text Box

Type the path and name of your existing workspace.

New Workspace Name Text Box

Tyoe the path and new name for your workspace.

Buttons
OK Applies the changes and closes the dialog box.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
