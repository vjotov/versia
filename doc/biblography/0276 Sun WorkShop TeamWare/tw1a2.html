View Menu

The View menu in the Configuring Window provides the following commands, which are choices that deal with how you view the workspaces. The options that are enabled have a small box to the left of them.
Vertical Orientation Sets vertical layout with parent at the top and children below.
Horizontal Orientation Sets horizontal layout with parent to the left and children branching to the right.
Full Names Displays the full path name of the workspaces.
Short Names Displays the working directory name of the workspace.
Descriptive Names Displays the name you have defined in Workspace Properties. (See Giving a Workspace a Descriptive Name)
Refresh Updates the workspace pane.
