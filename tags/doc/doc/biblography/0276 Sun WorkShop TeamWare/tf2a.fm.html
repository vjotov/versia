Freezepointing Defined Freezepointing Window

Freezepointing enables you to create freezepoint files from Sun WorkShopTM TeamWare workspaces. Freezepointing files list the default deltas for each file in the workspace. Later, you can re-create a version of the file or entire workspace by extracting information from the freezepoint file.(See Creating a Freezepoint)

You have the option of recreating Sun WorkShop TeamWare workspaces from the freezepoint file. You can extract the complete set of frozen files or a subset of the files. By default, the recreated workspace does not contain any SCCS history files. It contains only the files from which the orginal files were created. (See Recovering Files From a Freezepoint File)

You can configure Sun WorkShop TeamWare to create a freezepoint every time you perform a certain transactions, like bringovers and putbacks. (See Automatically Generating Freezepoints)

Note - Because a freezepoint file is only a list of differences, you must have the original workspace (or its parent or child) to extract from a freezepoint file.
