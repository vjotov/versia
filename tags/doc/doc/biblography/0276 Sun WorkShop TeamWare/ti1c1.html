Changing Your Editor

Related Topics:

Editing a File

To change your default editor:

1. In the Configuring Window, choose TeamWare Versioning.

The Versioning window appears.

2. In the Versioning Window, choose View Options.

3. In the Options Dialog Box, select your editor of choice: Textedit, Emacs, Emacsclient or Other.

4. If you choose Other, type the path for your editor in the box that appears.
Note - Not all editors are available in all locales.
