Save As Dialog Box

Use the Save As dialog box to save a file with a new name. This is useful if you want to save more than one version of a merged file. To access the Save As dialog box, in the Merging Window, choose File Save As.

This dialog box contains the following components:

Directory Pane

Displays the current directory.

Files Pane

Displays the files in the current directory. Clicking on a filename in this pane will put a filename into the name box.

Caution - Selecting an existing filename from this pane and then clicking OK will overwrite an existing file without warning.

Name Box

Type the name you want to use for the file.

Buttons
OK Saves the file and closes the dialog box.
Update Refreshes the list in the Files pane. The button only works if you have a directory selected in the Files pane, not an individual file.
Cancel Closes the dialog box without saving the file.
Help Displays help for this dialog box.
