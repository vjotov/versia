Welcome to Sun WorkShop TeamWare

When you work on a large project with many team members and many more files, it is hard to keep track of the files. Source code files change when you add new features, fix bugs and release new versions of the software. Sun WorkShopTM TeamWare configuration management tools help you manage projects by keeping track of files and preventing users from writing over each other's changes. With Sun WorkShop TeamWare, you have continuous access to all files in a project, and the tools to maintain the integrity of the files.

With Sun WorkShop TeamWare configuration management tools, you can:

  • Set up private workspaces for each user (see Configuring Defined)
  • Work concurrently on the same files (see Versioning Defined)
  • Easily see when files are checked out (see Versioning Defined)
  • Resolve changes to files with side-by-side, color-coded Merging (see Merging Defined)
  • View the history of a file, including merged and unmerged deltas, or changes, in File History (see Viewing File History)
  • View the history of a workspace in the Workspace History Viewer (see Viewing and Customizing Workspace History)
  • Save a snapshot of files using Freezepointing (see Freezepointing Defined)
