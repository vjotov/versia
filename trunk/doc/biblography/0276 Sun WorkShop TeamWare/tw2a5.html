Undo Tab

Related Topics:

Reversing a Bringover or Putback

Use the Undo tab in the Transactions Dialog Box to undo, or reverse, the most recent bringover or putback in the workspace specified in the Workspace Directory box. To open the Undo dialog box, in the Configuring Window, choose Actions Undo.

In this tab are the following components:

Workspace Text Box

Shows the name of the child directory from the last transaction.

Options Check Boxes
Preview Previews by running a bringover without actually transferring files.
Quiet Suppresses the output of status messages to the Transaction Output dialog box.

Buttons
Undo Starts the reversal of the last transaction.
Show Output Opens the Transaction Output dialog box.
Close Closes the Undo Create dialog box.
Help Displays online help for this tab.
