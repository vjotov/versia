Freezepointing Window

In the Freezepointing window, you'll find the following components:
File Menu Provides the Exit command for quitting Freezepointing.
View Menu Provides the Show Output command, where you can view and save status and error messages.
TeamWare Menu Provides commands to start other Sun Workshop TeamWareTM services.
Help Menu Provides commands to display help.
Freezepointing Creation Tab Displays the freezepoint creation parameters.
Freezepointing Extraction Tab Displays the freezepoint extraction parameters.
