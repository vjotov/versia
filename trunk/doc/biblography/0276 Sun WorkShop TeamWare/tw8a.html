Delete Workspaces Dialog Box

Related Topics:

Deleting Workspaces

Use the Delete Workspaces dialog box to confirm deletion of workspaces. To open the Delete Workspaces dialog box, in the Configuring Window, choose Workspace Delete.

This dialog box contains the following items:

Workspace Pane

Lists the workspaces you want to delete.

Buttons
Delete Sources and Codemgr_wsdata Directory button Deletes everything in the workspace: source files and codemgr_wsdata directory.
Delete Codemgr_wsdata Directory Only button Deletes only the codemgr_wsdata directory, so your files are still intact, but they are no longer in a Sun WorkShopTM TeamWare workspace.
OK Deletes the workspaces and closes the dialog box.
Cancel Closes the dialog box without deleting workspaces.
Help Displays online help for this dialog box.
