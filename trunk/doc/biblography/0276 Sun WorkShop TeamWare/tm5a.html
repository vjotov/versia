Find Dialog Box

Use the Find dialog box to find text strings in files. To access the Find dialog box, in the Merging Window, choose Navigate Find.

This dialog box contains the following components:

Pane Selector

Allows you to select which pane you want to search: Left, Right or Bottom.

Find Box

Type a text string you want to search for.

Case Sensitive Check Box

Check this box if you want the search to be case sensitive (for example, "ARG" matches ARG but not Arg or arg.)

Find Backwards Check Box

Check this box if you want to search up from your current location in the file. The default is to search down from your current location.

Buttons
Find Starts the search.
Clear Clears the Find Box.
Close Closes the dialog box without starting a search.
