General Tab

Use the General tab in the Options Dialog Box to modify what you view in the main filelist and History Graph, modify the mouse double-click action, and change your default editor. To open the General Options tab, in the Versioning Window, choose View Options and select the General tab.

This ab has the following components:

Options Buttons
Version Control Currently only accepts SCCS.
Main Filelist List all files under Version Control:Shows all files under SCCS control. This is the default.
List only files which are checked out.
List only files which are checked in.
Double Click Action Show File History: Brings up the versioning window for the selected file. The default.
Toggle File State: Double-clicking on a file checks out the file.
Confirm Double Click Check Out Check Box: Double-clicking on a file displays a confirmation window before checking out the file.
History Graph Sets which items appear in the History Graph pane in the File History Window: Show Removed Deltas or Show Branch Closure.
History Information Sets the default for History Information. Per-SID Information: Information from SCCS IDs (SIDs) is displayed panes labeled Delta Details and Delta Comments.
Entire History File: Instead of the Delta Details and Delta Comments panes, Versioning displays the entire file history.
File History Command When Show Entire File History is selected, this option is enabled, and you can type a command in the box. The default command is sccs prt.
Editor of Choice

Sets the editor that is launched when you edit a file. If you choose "Other," enter the path to your editor on the box to the right.

Buttons
OK Applies the changes and closes the dialog box.
Apply Applies the changes without closing the dialog box.
Reset Cancels current changes.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
