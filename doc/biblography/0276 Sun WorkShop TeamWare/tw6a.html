Create Workspaces Dialog Box

Use the Create Workspaces dialog box to create workspaces. To open the this dialog box, in the Configuring Window, choose File Create Workspaces.

This dialog box contains the following items:

Workspace Directory Text Box

Lets you type the path name of the new workspace.

Butttons
OK Applies the changes and closes the dialog box.
Cancel Closes the dialog box without making changes.
Help Displays the help for this dialog box.
