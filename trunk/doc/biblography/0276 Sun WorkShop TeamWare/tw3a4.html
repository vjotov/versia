Giving a Workspace a Descriptive Name

Use the Description tab to give a descriptive name to a workspace. You can also enter a long description for the workspace. To open the Description tab, in the Configuring Window, choose Workspace Properties and select the Description tab.

This tab contains the following items:

Name Text Box

Type a descriptive name for the workspace.

Description Text Box

Type a detailed description for the workspace.

Load Button

Loads the descriptive name and workspace from the Codemgr_wsdata/description file.
