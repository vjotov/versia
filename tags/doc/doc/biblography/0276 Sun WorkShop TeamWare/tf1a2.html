Use the Freezepointing Extraction tab to extract files from a freezepoint file.

This tab has the following components:

Full Extract Radio Button

Select to extract the complete set of frozen files.

Partial Extract Radio Button

Select to extract a subset of the frozen files.

Create Workspace Check Box

Select to create a Sun WorkShopTM TeamWare workspace from the frozen files.

Freezepoint File Box

Type the absolute path name of an existing freezepoint file.

Workspace Box

Type the name of the source workspace.

Destination Directory Box

Contains a list of files and directories that will be preserved in the freezepoint file.

Extract Freezepoint Sources Radio Buttons
Use default from freezepoint file Uses the path name of the source workspace.
You Specify Lets you type a workspace path name.
Show default and comments Shows the default workspace and the comments from freezepoint.out file.

Comments Pane

Displays the comments from the freezepoint file.

Extract Button

Starts the file extraction, displays the Freezepoint Output box with messages about the extraction.
