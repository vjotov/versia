Command Buttons

The command buttons in in the Merging Window help you navigate among differences and accept them for integration into the compilation file in the Results pane. The buttons above the Child pane and the Parent pane are identical; each button applies to the pane directly below. The buttons are:
Accept Resolves the current difference.
Next Moves to the next unresolved difference.
Accept & Next Resolves the current difference and advances to the next unresolved difference.
Up Arrow Moves to the previous difference whether resolved or unresolved.
Down Arrow Moves to the next difference whether resolved or unresolved
