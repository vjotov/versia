History Viewer Options Tab

Use the History Viewer Options tab to set defaults for the Workspace History Viewer. You can select multiple check boxes, but you can only choose one from the set of radio buttons. To open the History Viewer Options tab, in the Configuring Window, choose Options Configuring and select the History Viewer tab.

This tab contains the following items
Display check boxes Determine which items will appear in the Transaction Information pane of the History Viewer. You can select:
Time Date and time when the operation started.
Operation Name of the TeamWare operation (Bringover, Freezepoint, Putback, etc.).
User User name.
Host Name of the system where the operation originated.
TW Release TeamWare version (1.0.4, 2.0, 2.1)
Exit Status Result of the operation (Blocked, Conflict, Failure, Success).
Sort By radio buttons Determines how the workspaces in the History Viewer are sorted. You can choose:
Time Chronologically by date and time (default). Oldest entries appear at the top of the list.
Operation Alphabetically by name of the TeamWare operation.
User Alphabetically by user name.
Host Alphabetically by name of the system where the operation originated.
TW Release Numerically by TeamWare release number.
Exit Status Alphabetically by exit status (Blocked, Conflict, Failure, Success).
Filter check boxes Determine which filtering options appear in the History Viewer dialog box. There, you can select them and apply them to the Transaction Information pane. Filters you can include are:
Operation TeamWare operation (Bringover, Freezepoint, Putback, etc.)
User User name.
Host Name of the system where the operation originated.
TW Release TeamWare version (1.0.4, 2.0, 2.1).
Exit Status Result of the operation (Blocked, Conflict, Failure, Success).
