Reversing a Bringover or Putback

Related Topics:

Integrating Changes by Putting Back Files Resolving Differences Between Files Using Merging

To reverse a transaction:

1. In the Configuring Window, select the workspace that you have just changed.

2. Choose Actions Undo. This displays the Undo Tab.

If you have just executed a bringover or putback and you are already in the Transaction dialog box, select the Undo tab at the top of the Transactions Dialog Box.

3. Click Undo.

The workspace is restored to its previous state.

Note - This procedure works only for your most recent transaction.
