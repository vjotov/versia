Integrating Changes by Putting Back Files

Related Topics:

Resolving Differences Between Files Using Merging Reversing a Bringover or Putback

To put back a file to a parent workspace after you have made changes:

1. In the Configuring Window, choose Actions Putback.

2. In the Putback Tab, select the files you want to put back.

3. Click in the Comments box and type a comment.

Remember, depending on how your administrator has set up e-mail notification, many people may see this comment.

4. Click Putback.

5. The Transactions Output window is displayed and shows information about the putback. Click Close to dismiss this window.
