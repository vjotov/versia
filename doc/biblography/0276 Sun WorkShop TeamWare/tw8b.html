Edit Locks Dialog Box

Related Topics:

Removing Workspace Locks

Use the Edit Locks dialog box to remove locks from workspaces. To open the Edit Locks dialog box, in the Configuring Window, choose Workspace Edit Locks.

This dialog box contains the following items:

Workspace Text Box

Displays the name of the selected workspace, or you can type the name of a workspace.

Lock Information Pane

Lists the locks for the workspace.

Buttons
Delete Deletes the selected lock from the workspace.
Undo Restores the most recently removed lock.
OK Applies the changes and closes the dialog box.
Apply Applies the changes without closing the dialog box.
Reset Reverses the most recent change.
Cancel Closes the dialog box without making changes.
Help Displays online help for this dialog box.
