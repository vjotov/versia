Moving or Renaming Files

Related Topics:

Checking In a File

Caution - Do not use operating system commands to move or rename files under Sun WorkShopTM TeamWare control. Use this procedure instead. This procedure maintains the history of the file.

To move or rename a file:

1. In the Versioning Window, navigate to the workspace that contains the file you want to move or rename.

2. Click the file to select it. You can Shift-click to select multiple files.

3. Choose Workspace Move Files.

4. In the dialog box, type in the new name or location for your file.

If you want to give the file a new name, type the new name. If you want to move the file, type the path for the new location.

5. Click OK.
