Add Files Dialog Box

Related Topics:

Updating the Files in Your Workspace Integrating Changes by Putting Back Files

Use the Add Files dialog box to select which files you want to include in a bringover or putback. To open the Add Files dialog box, in the Configuring Window, choose Actions Bringover Create/Bringover Update/Putback and click Add.

This dialog box contains the following items:

Directory Display

Shows the currently selected directory.

File Pane

Lists the contents of the currently selected directory. Files that are checked out have a red check mark on them.

Name Text Box

Lets you enter a specific path or file name or change the default filter (for files only). For example, to list only header files for a directory, type *.h. Type . to specify the entire workspace (all files).

Buttons
Add Files Adds the selected files to the File pane.
Load Directory Displays the contents of the directory entered in the Name text box.
Close Closes the dialog box without making changes.
Help Displays online help for this dialog box.
