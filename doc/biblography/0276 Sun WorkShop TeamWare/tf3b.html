Show Output Dialog Box

The Show Output dialog box appears automatically during a Freezepointing transaction. Or, you can choose View Show Output in the Freezepointing Window to show output from the most recent Freezepointing transaction.

This dialog box contains the following items:

File Menu

Allows you to save the information from the Output dialog box to a text file.

Edit Menu

Clears the Output dialog box window.

Output Pane

Displays the output from the Freezepointing transaction, including the command initiated, the workspace, the files, and any warnings or errors.

Buttons
Stop Stops the tranaction.
Close Closes the dialog box.
Help Displays help for this dialog box.
