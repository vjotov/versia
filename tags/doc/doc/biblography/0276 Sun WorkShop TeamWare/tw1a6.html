TeamWare Menu

The TeamWare menu in the Configuring Window provides the following commands:
Configuring Opens Configuring Window. See Configuring Defined.
Freezepointing Opens the Freezepointing Window. See Freezepoint Basics.
Building Opens the Building window.
Merging Opens the Merging Window. See Resolving Differences Between Files Using Merging.
Versioning Open Versioning Window. See Basic Concepts of TeamWare File Management.
