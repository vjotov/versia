Command Output Dialog Box

Use the Comand Output dialog box in the Versioning Window to view output from a failed transaction.

This dialog box contains the following components:

Display Check Boxes
Commands Shows commands in the Output pane.
Results Shows results in the Output pane.
Errors Shows errors in the Output pane.

Output Pane

Displays output from a failed transaction.

Buttons
Clear Clears the Output pane.
Close Closes this dialog box.
Help Displays help for this dialog box.
