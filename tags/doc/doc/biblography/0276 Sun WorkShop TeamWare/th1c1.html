Adding and Removing Transaction Information

To add or remove transaction information from the Workspace History Viewer:

1. In the Configuring Window, click on a workspace to select it.

2. Choose Workspace View History.

3. In the Workspace History Viewer Window, choose View Options.

4. In the Display section, select which transaction information you want to appear in the Workspace History window. Click a check box to include the transaction.

5. Click OK.
