Use the Workspace Properties dialog box to set properties for a specific workspace, such as putback validation, e-mail notification, workspace description and auto-freezepointing. To open the Workspace Properties dialog box, in the Configuring Window, choose Workspace Properties.

This dialog box contains the following components:

Tabs
Access Control Tab Allows you to grant or deny permission for individuals and groups to do transactions in a workspace.
Putback Validation Tab Allows you to protect a workspace by allowing only putbacks and requiring a Integration Request ID from users.
Notification Tab Allows you to set up automatic e-mail notification triggered by transactions in a workspace.
Description Tab Allows you to give a workspace a descriptive name.
Freezepointing Tab Allows you to set up auto-freezepointing.

Buttons
OK Applies the change and closes the dialog box.
Apply Applies the change without closing the dialog box.
Reset Reverses the most recent change.
Cancel Closes the dialog box without applying changes.
Help Displays online help for this dialog box.
