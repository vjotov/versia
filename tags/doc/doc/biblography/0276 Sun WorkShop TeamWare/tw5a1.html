Load Children Dialog Box

Use this dialog box to load children of the selected workspace. To open the Load Children dialog box, from Configuring, choose Workspace Load Children.

This dialog box contains the following items:

Directory Box

Type the name of the workspace here.

Select Child Workspaces To Load Pane

Lists the children of the current workspace. Click a workspace to load.

Buttons
Select All Selects all the workspaces in the pane.
Deselect All Deselects all the workspaces in the pane.
Load All Loadsall the workspace into the Configuring window.
Load Selected Loads the selected workspace into the Configuring window.
Cancel Closes the dialog box without making changes.
Help Opens help.
