Goto Line Dialog Box

Use the Goto Line dialog box to move to a specific line in one of the Merging panes. This dialog box is displayed until specifically dismissed. To access the Goto Line dialog box, in the Merging Window, choose Navigate Goto Line.

This dialog box contains the following components:

Goto Line Box

Type the line number.

Pane Selector

Allows you to select which pane you want to move around in: Left, Right or Bottom.

Buttons
Goto Moves to the cursor to the line in the Goto Line box.
Clear Clears the Goto Line Box.
Close Closes the dialog box.
