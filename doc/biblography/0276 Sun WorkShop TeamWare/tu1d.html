New to Sun WorkShop TeamWare?

Related Topics:

Basic Concepts of Sun WorkShop TeamWare

These steps assume you are new to a workgroup that is already using Sun WorkShopTM TeamWare. If a TeamWare environment has not already, been set up, see Setting up a TeamWare Environment.

The workflow of Sun WorkShop TeamWare is:

1. Find the parent workspace and load it.

Configuring: File Load Workspace

2. Create a child workspace.

Configuring: Actions Bringover Create

3. Check out a file.

Versioning: Commands Check Out

4. Edit the file.

Your favorite editor

5. Check in the file.

Versioning: Commands Check In

6. Put back the file to the parent workspace.

Configuring: Actions Putback
