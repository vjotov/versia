Moving or Renaming Files Deleting and Renaming Files Issues

Caution - Use this procedure to move or rename files, rather than with operating system commands. This procedure will maintain the relationships and history of the file.

To move or rename a file:

1. In the Versioning Window, navigate to the workspace that contains the file.

2. Click on the file to select it. You can Shift-click to select multiple files.

3. Choose Workspace Delete Files.

4. In the dialog box, type in the new name and/or location for your workspace.

If you want to give the file a new name, type the new name. If you want to move the file, type the path for the new location.

5. Click OK.
