View Menu

The View Menu in the Versioning Window contains the following commandsf.
Update Rereads the current directory and shows the latest files and directories in the Directory pane.
List Checked Out Shows only checked-out files in the Directory pane, but all directories.
List Checked In Shows only checked-in files in the Directory pane, but all directories.
List All Shows all files and directories in the Directory pane.
Command Output Opens the Command Output Dialog Box, where you can view messages from a failed transaction.
Options Opens the Options Dialog Box, where you can set Versioning options.
