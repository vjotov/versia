Commands Menu

The Commands Menu in the Versioning Window contains the following commandss.
Check Out Checks out the selected file or files, see Checking Out a File.
Check Out and Edit Checks out the selected file and opens in a text editor. To change your text editor, see Changing Your Editor.
Check In Opens the Check In Dialog Box, where you can type a comment before checking in the selected file or files.
Check In New Opens the Check In New Dialog Box, where you can select files to be put under SCCS control.
Uncheckout Reverses the last checkout for a selected file by reverting to the previous state of the file.
