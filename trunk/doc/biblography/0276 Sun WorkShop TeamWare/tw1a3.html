Workspace Menu

The Workspace menu in the Configuring Window provides the following commands:
Load Parent Loads the parent workspace of the workspace selected in the Workspace pane.
Load Children Opens the Load Children Dialog Box, where you can select Child workspaces to load.
Create Child Opens the Create Child Workspace Dialog Box, where you can type the names of the parent workspace and the new child workspace.
Delete Opens the Delete Workspaces Dialog Box, where you can delete the selected workspace(s).
Edit Locks Opens the Edit Locks Dialog Box, where you can remove locks created by an interrupted transaction.
Rename Opens the Rename Dialog Box, where you can rename the selected workspace.
Reparent Opens the Reparent Dialog Box, where you can reparent the selected workspace
Update Nametable Opens the Update Nametable Dialog Box, where you can reparent the selected workspace
View History Opens the Workspace History Viewer Window, where you can view the history of the selected workspace.
Properties Opens the Workspace Properties Dialog Box, where you can set access control, putback validation, email notification, workspace descriptions and autofreezepointing.
