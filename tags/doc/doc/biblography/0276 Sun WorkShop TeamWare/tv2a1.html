View Menu

The View menu in the File History Window provides the following commands:
Diff in Merging Window Starts FileMerge to show the differences between the selected deltas.
Diff in Text Window Shows the differences between select deltas in editor's window in "diff" format.
Diff with Context Shows the differences between select deltas in editor's window in "diff -c" format (with context).
Show File Contents Shows the contents of the selected delta in editor's window.
Textual History Shows the entire history of the file.
Selected Delta Info Shows detailed information about the selected delta.
Default Delta Selects the default delta on the graph.
Branch Closure Toggles on/off the dashed arrows that show branch closures.
Removed Deltas Toggles on/off removed (crossed-out) deltas.
Mark Unmerged Deltas Marks unmerged deltas on the graph with a rectangle.
