How Can Sun Workshop TeamWare Help Me?

Related Topics:

Hints from Other Users

With Sun WorkShopTM TeamWare configuration management tools, you can:

  • Set up private workspaces for each user (see Configuring Defined)

  • Work concurrently on the same files (see Versioning Defined)

  • Easily see files when are checked out (see Versioning Defined)

  • Resolve changes to files with side-by-side, color-coded Merging (see Merging Defined)

  • View the history of a file, including merged and unmerged deltas in File History (see Viewing File History)

  • View the history of a workspace in the History Viewer (see Viewing and Customizing Workspace History)

  • Save a snapshot of files with Freezepointing (see Freezepoint Basics)
