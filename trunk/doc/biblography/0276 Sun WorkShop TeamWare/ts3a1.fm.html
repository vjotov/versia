Setting Global Permissions

Setting Individual or Group Permissions

You can grant or deny access to workspaces on a global, group or individual basis.

To specify global permissions:

1. In the Configuring Window, load the workspace by choosing File Load Workspaces.

2. Select the workspace by clicking on it.

3. Choose Workspace Properties.

4. Select the Access Control Tab in the Workspace Properties Dialog Box.

5. Double-click an operation to bring up the Access Control Dialog Box for that operation.

6. Select the type of permission you want to allow:

  • None: No users have permission

  • All: All users have permission

7. Click OK.
